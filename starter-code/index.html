Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku, Pexeso, or Pairs, is a card game in which all of the cars are laid facedown on surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

The Game

Concentration is a fun game for all ages, especially for those looking to improve their memory skills.
